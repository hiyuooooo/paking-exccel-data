# Transaction Manager - Windows Package

## Quick Start
1. Make sure Node.js is installed from https://nodejs.org/
2. Double-click start.bat (or run start.sh on Linux/Mac)
3. Browser will open automatically to http://localhost:3000
4. Start using Transaction Manager!

## What's Included
- dist/server/production.mjs - Server application
- dist/spa/ - Web application files
- start script - Easy launcher

## Requirements
- Node.js v16 or higher
- Modern web browser
- Windows 10/11 (for .bat files)

## Features
- Complete transaction management
- Customer database
- Data import/export (PDF, Excel)
- Monthly reports and summaries
- Automatic data backup
- Runs entirely offline

## Support
If you have issues:
1. Make sure Node.js is installed
2. Check if port 3000 is available
3. Try running: node dist/server/production.mjs
4. Open browser manually to http://localhost:3000
